# Etc

* [Presentation Of Interactive Normalform Transformations](point.html) (point)  
Tool I developed as part of teaching a database course to come up with functional dependencies in various normal forms.  The source code can be found [here](https://github.com/cem-okulmus/point).
* [shaWell](https://github.com/cem-okulmus/shawell/)  
A [SHACL](https://www.w3.org/TR/shacl/) validator that supports the 1.0 SHACL Core standard, and has support for recursive shapes. It follows the well-founded semantics for SHACL, as proposed [in our paper](https://dl.acm.org/doi/10.24963/kr.2024/52).
